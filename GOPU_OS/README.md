# GOPU OS editable map

This folder is the human-editable map for GOPU OS wording, module names, and operating configuration.

Start here when you want to change visible business labels without searching through the full app:

- `cto/labels.js` - CTO dashboard and provider vault labels.

The production app still keeps workflow logic in `src/`, backend/server logic in `server/`, `api/`, and shared server libraries in `lib/`.
